# Enterprise Software Catalog

This catalog lists all products available under our enterprise licensing program.
Pricing is shown per seat per month. Volume discounts apply for orders over 500 seats.

## Security Suite

The Security Suite covers endpoint protection, identity management, and network monitoring.
All products are SOC 2 Type II certified and support SSO via SAML 2.0 and OIDC.
